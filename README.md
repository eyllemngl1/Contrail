## 🚀 What is CONTRAIL?

CONTRAIL is an aviation intelligence dashboard that monitors 10 major airlines in real-time, analyzes news with AI, and delivers actionable signals to investors and aviation professionals.

**Core Features:**
- 📡 Real-time news monitoring from 10 global airlines
- 🤖 AI-powered news scoring (1-10) and Turkish summarization via Groq API
- 📊 Live market data (stock prices, Brent, EUR/USD, USD/TRY, DXY)
- 🚨 Telegram alerts for high-impact news (score 7+)
- 📰 Automated Intelligence Brief every 2 hours
- 🌍 LCC vs FSC sentiment comparison
- ⚡ Mayday Alerts for crisis signals
